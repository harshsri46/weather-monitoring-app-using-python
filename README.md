Weather Monitoring App Using Python

Project Description
This is a command-line weather application built using Python. It fetches real-time weather data
from the OpenWeatherMap API based on the city name entered by the user.

Technologies Used
• Python 3.x
• requests library
• OpenWeatherMap API
• JSON data parsing

How It Works
• User enters a city name.
• Python script creates API request URL.
• HTTP GET request is sent using requests library.
• API returns JSON response.
• Weather description, temperature, feels-like temperature and humidity are displayed.

Sample Output

Enter city name:
Delhi
Displaying weather report of Delhi
clear sky
Current temperature is 28
Current temperature feels like 30
Humidity is 60
